# Online Auction System (Django)

## 📌 Project Description
This is a Django-based Online Auction System developed as a college project.  
It allows users to register, login, list products for auction, and place bids.

---

## 🚀 Features
- User Registration & Authentication
- Product Listing for Auction
- Bidding System
- Admin Panel
- Secure Login System

---

## 🛠 Tech Stack
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

---

## ⚙️ Installation & Setup

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Apply migrations:
   python manage.py migrate
4. Run server:
   python manage.py runserver
5. Open:
   http://127.0.0.1:8000/

---

## 📂 Project Structure
- project_folder → Main Django project
- app_folder → Application logic
- templates → HTML templates
- static → CSS & JS files
